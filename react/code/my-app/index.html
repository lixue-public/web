what's react

定义

rect是一个 可复用 可聚合 的web 组件的js库 ？？ 为什么说它可复用、可聚合

特点

组件化:js逻辑和HTML标签紧密相连，容易理解，包装整体

单向数据流： react渲染方式 用户输入 从API获取数据 （哪些API呢） 将数据传给顶层组件（组件嵌套？？哪个是顶层？哪个是父子 react将每个组件渲染出来，而不是某个具体的DOM操作 这种渲染方式的好处 组件功能明确 结果可以预测（逻辑控制了状态） component类似独立函数 可以接收参数（props) 可复用 可传递 可返回结果（渲染效果）

虚拟的DOM树 Js很快，但是DOM操作慢 react如何更新、进行DOM操作 React根据当前props和states 重建DOM树 找到与上个DOM树的diff 特有的算法，计算how进行DOM更新效率高 从操作队列批量 更新DOM操作
